API Reference

‚ÑπÔ∏è Base URL: https://og-miner-api.herokuapp.com/

1. Extract Metadata

POST /v1/extract

Main endpoint to scrape and parse a URL.

curl -X POST "https://og-miner-api.herokuapp.com/v1/extract" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "url": "https://github.com",
           "enable_javascript": false,
           "force_refresh": false
         }'

Example Response (200 OK)

{
  "meta": {
    "url": "https://github.com",
    "domain": "github.com",
    "latency_ms": 245
  },
  "data": {
    "title": "GitHub: Let's build from here",
    "description": "GitHub is where over 100 million developers shape the future of software...",
    "image": "https://github.githubassets.com/images/modules/site/social-cards/github-social.png",
    "favicon": "https://github.com/favicon.ico",
    "site_name": "GitHub",
    "oembed": { ... }
  }
}

2. Take Screenshot

POST /v1/screenshot

Capture pixel-perfect screenshots.

curl -X POST "https://og-miner-api.herokuapp.com/v1/screenshot" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "url": "https://stripe.com",
           "full_page": true,
           "dark_mode": true
         }'

Example Response (200 OK)

{
  "screenshot": "iVBORw0KGgoAAAANSUhEUgAABAAAAAMgCAYAAAC...", // Base64 encoded PNG
  "meta": {
    "width": 1280,
    "height": 4500,
    "size_kb": 1240
  }
}

3. Batch Processing

POST /v1/batch/extract

Process up to 50 URLs in parallel.

curl -X POST "https://og-miner-api.herokuapp.com/v1/batch/extract" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "urls": ["https://google.com", "https://apple.com"]
         }'

Example Response (200 OK)

{
  "total": 2,
  "successful": 2,
  "failed": 0,
  "results": {
    "https://google.com": { "status": "success", "data": { "title": "Google", ... } },
    "https://apple.com": { "status": "success", "data": { "title": "Apple", ... } }
  }
}

4. Image Proxy

GET /v1/image

Resize and cache images.

curl "https://og-miner-api.herokuapp.com/v1/image?url=https://example.com/img.png&width=300"

5. Error Codes

  • 200 OK: Success.
  • 400 Bad Request: Invalid URL or parameters. (Check 'code': 'invalid_request')
  • 403 Forbidden: Invalid API Key/Secret.
  • 429 Too Many Requests: Rate limit exceeded.
  • 502 Bad Gateway: Upstream connection failure. (Check 'code': 'connection_error')
  • 504 Gateway Timeout: Upstream timeout. (Check 'code': 'timeout')

Guides & Use Cases

Handling Single Page Apps (SPAs)

Standard scrapers fail on sites like Instagram or TikTok. OG Miner solves this with the enable_javascript flag.

‚ö†Ô∏è Note: Headless requests take longer (2-5s).

Geo-Targeting (Beta)

Preview how content looks from different regions by using the country parameter (e.g., "US", "EU"). 

curl -X POST "https://og-miner-api.herokuapp.com/v1/extract" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "url": "https://netflix.com",
           "country": "US"
         }'

Bring Your Own Proxy (BYOP) (Beta)

Need to access a site behind a corporate firewall or use a specific residential IP? Pass your proxy URL in the proxy field. 

curl -X POST "https://og-miner-api.herokuapp.com/v1/extract" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "url": "https://target-site.com",
           "proxy": "http://user:pass@residential-proxy.com:8000"
         }'

Authenticated Scraping

Scrape content behind a login wall by passing session cookies in the cookies object. 

curl -X POST "https://og-miner-api.herokuapp.com/v1/extract" \
     -H "Content-Type: application/json" \
     -H "X-RapidAPI-Proxy-Secret: YOUR_SECRET_KEY" \
     -d '{
           "url": "https://linkedin.com/in/sasmithakg",
           "cookies": { "li_at": "AQED..." }
         }'
